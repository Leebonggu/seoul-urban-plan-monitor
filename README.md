# 서울 도시계획 결정고시 모니터

서울시 도시계획 결정고시를 매일 자동으로 수집하고, **2040 서울플랜 중심지체계**에 따라 분류하여 보여주는 서비스입니다.

## 왜 만들었나

서울시는 재개발·재건축·용도변경 등 도시계획이 결정되면 **결정고시**를 발표합니다. 이 고시문은 [서울도시공간포털](https://urban.seoul.go.kr)에 올라오지만, 매번 직접 확인해야 하고 데이터를 축적하거나 비교·분석할 수 없습니다.

이 서비스는 고시문을 **자동 수집·축적**하고, 2040 서울플랜이 지정한 **22개 핵심 중심지**(3도심·7광역중심·12지역중심)와 매칭하여 어떤 지역에서 어떤 변화가 일어나고 있는지 한눈에 파악할 수 있게 합니다.

## 무엇을 볼 수 있나

- **지도** — 서울 22개 중심지별 고시문 현황을 지도에서 확인
- **최신 고시문** — 매일 새로 올라오는 고시문을 중심지 태그와 함께 조회
- **필터** — 중심지 등급(도심/광역/지역), 개별 중심지, 키워드로 관심 고시문만 추출
- **추이 분석** — 월별 고시문 추이, 유형 분포, 중심지별 랭킹
- **원문 바로가기** — 고시문 원문(PDF)과 상세 페이지 링크 제공

## 2040 서울플랜 중심지체계

| 등급 | 중심지 |
|------|--------|
| **도심** (3) | 한양도성, 여의도·영등포, 강남 |
| **광역중심** (7) | 용산, 청량리·왕십리, 창동·상계, 상암·수색, 마곡, 가산·대림, 잠실 |
| **지역중심** (12) | 동대문, 망우, 미아, 성수, 신촌, 마포, 공덕, 연신내·불광, 목동, 봉천, 사당·이수, 수서·문정, 천호·길동 |
