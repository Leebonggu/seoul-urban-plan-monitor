## v2 로드맵 (우선순위순)

### Phase 1: 체류시간 & 내부 링크 강화

#### 1. 관련 고시문 추천
- **효과:** 내부 링크 밀도 증가 + 체류시간 증가 (구글 랭킹 핵심 요소)
- **구현:** 개별 고시문 페이지 하단에 같은 center_name 또는 location의 고시문 3-5개 표시
- **난이도:** 낮음
- **파일:** `web/src/app/gosi/[notice_code]/page.tsx`, `web/src/lib/data.ts`

#### 2. 지역별 허브 페이지
- **효과:** "강남구 도시계획", "영등포구 재개발" 같은 지역 검색 키워드 장악
- **구현:** `/area/[district]` 동적 라우트 — 해당 구의 고시문 목록 + 통계
- **난이도:** 중간
- **파일:** `web/src/app/area/[district]/page.tsx` 신규

#### 3. GosiTable "상세" 링크 내부화
- **효과:** 외부 이탈 방지, 페이지뷰 증가
- **구현:** "상세" 링크를 서울시 외부 페이지 대신 우리 `/gosi/...` 페이지로 변경
- **난이도:** 매우 낮음
- **파일:** `web/src/components/GosiTable.tsx`, `web/src/components/GosiList.tsx`

### Phase 2: 콘텐츠 볼륨 확대

#### 4. 주간 자동 요약 리포트
- **효과:** 반복 방문 유도 + "서울 도시계획 주간 동향" 키워드 확보
- **구현:** 매주 월요일 지난주 고시문을 LLM으로 요약 → JSON 저장 → 대시보드 + WP 자동 발행
- **난이도:** 중간
- **파일:** `weekly_report.py` 신규, GitHub Actions 스케줄 추가

#### 5. WP 키워드 태그 자동화
- **효과:** WP 태그 페이지가 SEO 롱테일 키워드 역할
- **구현:** LLM이 추출한 keywords를 WP Tags API로 자동 등록
- **난이도:** 낮음
- **파일:** `wp_publisher.py` 수정

### Phase 3: 알림 & 유입 채널

#### 6. 텔레그램/슬랙 알림
- **효과:** 실시간 알림으로 충성 사용자 확보
- **구현:** 키워드/지역 매칭 시 알림 발송
- **난이도:** 중간
- **파일:** `notifier.py` 신규

#### 7. 이메일 뉴스레터 구독
- **효과:** 재방문 트래픽 채널. 주간 요약 리포트를 이메일로 발송
- **구현:** 구독 폼 + 발송 시스템 (Resend/SendGrid)
- **난이도:** 중간~높음

#### 8. 핫스팟 분석 대시보드
- **효과:** "이번 주 주목 지역 TOP 5" — 차별화 콘텐츠
- **구현:** 최근 N일 고시문 집중도 분석 컴포넌트
- **난이도:** 중간
- **파일:** `web/src/components/HotspotAnalysis.tsx` 신규

### Phase 4: 장기

#### 9. 멀티 도시 확장
- 인천, 경기, 부산, 대구 크롤러 모듈화
- **난이도:** 높음

#### 10. 인스타그램 자동 포스팅
- Meta Graph API (비즈니스 계정 필요)
- **난이도:** 높음

#### 11. HWP 썸네일 지원
- LibreOffice 변환 (전체 게시물의 ~1.6%)
- **난이도:** 중간

#### 12. 레거시 코드 정리
- crawler.py, db.py 리팩토링 (현재 미사용)
- **난이도:** 낮음
